# DESIGN SYSTEM — RandomChessDebut · Тема «Полночь»

> Тема для игры при слабом освещении — глубокий угольно-серый фон с минимальным контрастом интерфейса. Максимальный комфорт для глаз поздно ночью. Никакого синего света, никаких ярких акцентов — только шахматы и тишина.

---

## 1. Цветовая палитра

### Акцентные цвета

| Токен                  | Hex         | Применение                                   |
|------------------------|-------------|----------------------------------------------|
| `--color-accent`       | `#A0A0A8`   | CTA-кнопки, активные состояния, прогресс     |
| `--color-accent-hover` | `#B8B8C0`   | Hover для акцентных кнопок                   |
| `--color-accent-muted` | `#A0A0A826` | Фон акцентных тегов, бейджей                 |
| `--color-gold`         | `#888890`   | Ходы, выделенные позиции, «правильно»        |
| `--color-gold-muted`   | `#8888901A` | Фон золотых бейджей                          |

### Нейтральная шкала — тема «Полночь»

| Токен        | Hex       | Применение                              |
|--------------|-----------|-----------------------------------------|
| `--gray-950` | `#050506` | Максимально тёмный фон, overlays        |
| `--gray-900` | `#0A0A0C` | Основной фон приложения                 |
| `--gray-850` | `#101012` | Фон карточек, модальных окон            |
| `--gray-800` | `#161618` | Вторичный фон, sidebar                  |
| `--gray-700` | `#222226` | Borders, разделители                    |
| `--gray-600` | `#303035` | Disabled borders, inert элементы        |
| `--gray-400` | `#606068` | Placeholder, subtitle                   |
| `--gray-200` | `#909098` | Вторичный текст                         |
| `--gray-50`  | `#C8C8D0` | Основной текст                          |

> **Принцип:** В теме «Полночь» основной текст намеренно приглушён до `#C8C8D0` вместо чисто белого — это снижает усталость глаз при работе в темноте.

### Семантические цвета

| Токен                   | Hex         | Применение                        |
|-------------------------|-------------|-----------------------------------|
| `--color-success`       | `#4A8A5A`   | Правильный ход, успех             |

| `--color-error`         | `#8A4A4A`   | Неверный ход, ошибка              |
| `--color-error-muted`   | `#8A4A4A26` | Фон error-уведомлений             |
| `--color-warning`       | `#8A7A3A`   | Предупреждение, подсказка         |
| `--color-info`          | `#4A6A8A`   | Нейтральная информация            |

### Цвета доски

| `--board-light`     | `#9E9E9E`                |

## 2. CSS-переменные

```css
[data-theme="midnight"] {
  --color-accent:       #A0A0A8;
  --color-accent-hover: #B8B8C0;
  --color-accent-muted: rgba(160, 160, 168, 0.15);
  --color-gold:         #888890;
  --color-gold-muted:   rgba(136, 136, 144, 0.1);

  --gray-950: #050506;
  --gray-900: #0A0A0C;
  --gray-850: #101012;
  --gray-800: #161618;
  --gray-700: #222226;
  --gray-600: #303035;
  --gray-400: #606068;
  --gray-200: #909098;
  --gray-50:  #C8C8D0;

  --color-success:       #4A8A5A;
  --color-success-muted: rgba(74, 138, 90, 0.1);
  --color-error:         #8A4A4A;
  --color-error-muted:   rgba(138, 74, 74, 0.15);
  --color-warning:       #8A7A3A;
  --color-info:          #4A6A8A;

  --board-light:     #9E9E9E;
  --board-dark:      #4A4A4A;
  --board-highlight: rgba(200, 200, 80, 0.5);
  --board-hint:      rgba(74, 138, 90, 0.4);
  --board-error:     rgba(138, 74, 74, 0.5);
}
```
