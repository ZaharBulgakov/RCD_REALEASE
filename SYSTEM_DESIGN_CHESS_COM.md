# DESIGN SYSTEM — RandomChessDebut · Тема «Chess.com»

> Тема вдохновлена цветовой палитрой самой популярной шахматной платформы в мире. Зелёные тона доски, кремовый и тёмный фон — всё это создаёт ощущение профессиональной среды, знакомой миллионам шахматистов.

---

## 1. Цветовая палитра

### Акцентные цвета

| Токен                  | Hex         | Применение                                   |
|------------------------|-------------|----------------------------------------------|
| `--color-accent`       | `#81B64C`   | CTA-кнопки, активные состояния, прогресс     |
| `--color-accent-hover` | `#97CC5E`   | Hover для акцентных кнопок                   |
| `--color-accent-muted` | `#81B64C26` | Фон акцентных тегов, бейджей                 |
| `--color-gold`         | `#C9A227`   | Ходы, выделенные позиции, «правильно»        |
| `--color-gold-muted`   | `#C9A2271A` | Фон золотых бейджей                          |
