# DESIGN SYSTEM — RandomChessDebut · Тема «Аметист»

> Тема вдохновлена глубоким фиолетовым цветом аметиста — камня, которому приписывали способность обострять интеллект и сохранять ясность ума. Элегантная фиолетовая палитра для тех, кто предпочитает эстетику над строгостью.

---

## 1. Цветовая палитра

### Акцентные цвета

| Токен                  | Hex         | Применение                                   |
|------------------------|-------------|----------------------------------------------|
| `--color-accent`       | `#7C5CBF`   | CTA-кнопки, активные состояния, прогресс     |
| `--color-accent-hover` | `#9370D8`   | Hover для акцентных кнопок                   |
| `--color-accent-muted` | `#7C5CBF26` | Фон акцентных тегов, бейджей                 |
| `--color-gold`         | `#A888D4`   | Ходы, выделенные позиции, «правильно»        |
| `--color-gold-muted`   | `#A888D41A` | Фон золотых бейджей                          |
